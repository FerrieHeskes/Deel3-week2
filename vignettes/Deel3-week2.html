Fatal car injuries within the USA

Ferrie Heskes

2017-06-10

This package contains several functions which can be used to get insight of the fatal injuries within the USA, e.g. :

  • gives a map of the fatal injuries per state per year
  • gives the total injuries within the USA per year per month

It uses the following external libraries:

dplyr tidyr maps graphics

Figures

Per state per year a map can be made with all the fatal injuries per year.
